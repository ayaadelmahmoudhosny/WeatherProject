Weather Forecast App

#Overview
Mobile application that displays the weather ( temperature,rain and humidty)  hourly along the day based on the location.

#Features


- Load main scereen 
- Temperature (C\F)
- Time format (12\24)
- Load weather data based on location
- Rain and humidity is displayed
- Generate Reports
